# C-8Puzzles
The program use A* search and two different heuristic functions to solve random 8 slide puzzles. h1() estimates the total number of unmatched between the current state and the goal state. h2() estimates the total Manhattan distance between the current state and the goal state. In this program, OpenGL and GLUT are used.
